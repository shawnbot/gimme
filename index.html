TMI: Tiled Map Images

TMI is a JavaScript library for creating HTML image maps from geographic feature data in GeoJSON format. It uses other, tried-and-true libraries (such as ModestMaps and Polymaps) to project geographic coordinates into screen coordinates, and can easily be backed onto other libraries that provide similar functionality.

The goal of TMI is to make linked maps a reality in both ancient and modern web browsers.

ModestMaps tiled backgrounds

TMI can overlay transparent image maps on top of non-interactive modestmaps-js maps. This example links neighborhood areas from the San Francisco Association of Realtors (SFAR) to the corresponding Google Maps search results.

Polymaps-supported Data Images

This example consists of three pieces:

  1. a static image of the Mission neighborhood in San Francisco (stolen from geoff) that stores the map projection info in HTML custom data attributes
  2. a GeoJSON representation of the Mission neighborhood polygon
  3. a stripped-down version of Polymaps for client-side coordinate projection (Note: no SVG is generated in this example.)
